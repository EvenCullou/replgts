### 默认信息
```
地址：replgts.xxxx.repl.co
端口：443
协议：vless 或 vmess
UUID：62690da3-0d4a-47c4-a0cf-191dc8a6c476
传输层安全：tls
传输协议：WebSocket
WebSocket 路径：/vless 或 /vmess
```

### 自定义 UUID

Replit 内点击  🔒Secrets 选项

设定 ```key``` 为 ```UUID```
设定 ```value``` 为 ```UUID 内容```

点击 Add new secret 保存

重启应用生效
